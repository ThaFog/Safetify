EnumResolver

  • EnumResolver<T>(definition: Array<string | number> | EnumType): Resolver<T>
    • Defined in resolvers/Enum.ts:35

    Resolves enum

    example 
    
// Array representation
EnumResolver([ 'option1', 'option2', 'option3' ]).resolve('option1');

// Object representation

// TypeScript's enum
EnumResolver(someEnum).resolve('option1');

// output will be the same as input

EnumResolver(someEnum).resolve('option4');
// output will be the first enum item, in this case 'option1'
